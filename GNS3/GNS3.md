### Some commands

| Команда                                    | Описание                                       |
| ------------------------------------------ | ---------------------------------------------- |
| ```?```                                    | Просмотр всех комманд                          |
| ```show hosts```                           | Просмотр хостов                                |
| ```Tab```                                  | Дозаполнить название комманды                  |
| ```history```, $\uparrow$, $\downarrow$    | Просмтреть историю                             |
| ```Ctrl A```                               | Сдвиг каретки в начало строки                  |
|                                            |                                                |
